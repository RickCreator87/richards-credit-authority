
⭐ 6. gdf loan repay — Record Profit-Based Repayment

Command
`
gdf loan repay gdp-loan-2026-02-06-01 \
  --amount 350 \
  --source profits \
  --note "January profit allocation"
`

What it does
- Updates ledger  
- Validates repayment source  
- Updates outstanding balance  
- Triggers repayment automation rules
